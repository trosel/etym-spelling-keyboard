Etymological English Keyboard

A QWERTY-shaped keyboard for an etymology-driven English orthography.

Substitutions

QWERTY positionTypes
Qæ / Æ
K’ (right single quote) for contractions
V· (middle dot) for morpheme boundaries

Mobile longpresses

Hold c for q/k. Hold b or f for v. Hold i, a, u, or e for diacritic alternates (į, ị; ạ, ą; ų; ẹ, œ).

Desktop access

Hold Right-Alt with a key to reach buried letters: RAlt+C = q, RAlt+Shift+C = k, RAlt+B = v, RAlt+I = į, etc. See the included README for the full table.
