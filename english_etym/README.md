# Etymological English Keyman Keyboard

A QWERTY-shaped keyboard for an etymology-driven alternative English orthography. Built for desktop (Windows/macOS/Linux) and mobile (iOS/Android) via [Keyman](https://keyman.com).

## Layout substitutions

The home row and most letters stay where they sit on QWERTY. Three positions are remapped:

| QWERTY key | Types | Codepoint |
|---|---|---|
| Q | **æ** / **Æ** | U+00E6 / U+00C6 |
| K | **’** | U+2019 (right single quote) |
| V | **·** | U+00B7 (middle dot / interpunct) |

## Mobile longpress alternates

| Hold this key | To get |
|---|---|
| c | q, k |
| b | v |
| f | v |
| i | į, ị |
| a | ạ, ą |
| u | ų |
| e | ẹ, œ |

## Desktop access (Right-Alt modifier)

Since desktops have no longpress, **Right-Alt + key** reaches buried letters and diacritic alts:

| Press | Outputs |
|---|---|
| RAlt + Q | q (and RAlt + Shift + Q = Q) |
| RAlt + K | k |
| RAlt + V | v |
| RAlt + C | q (RAlt + Shift + C = k) |
| RAlt + B / RAlt + F | v |
| RAlt + I | į (RAlt + Shift + I = ị) |
| RAlt + A | ạ (RAlt + Shift + A = ą) |
| RAlt + U | ų |
| RAlt + E | ẹ (RAlt + Shift + E = œ) |
